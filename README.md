# PHP & MySQL Student Management System

## Overview
This project is a simple full-stack CRUD web application developed using PHP and MySQL.  
It allows users to add, view, and delete student records.

## Features
- Add new students
- View student list
- Delete student records
- Database connectivity using MySQL
- Basic form validation

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- XAMPP (Apache Server)

## How to Run
1. Install XAMPP
2. Move project folder to htdocs
3. Create database in phpMyAdmin
4. Import SQL table
5. Run via localhost
